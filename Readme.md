### Descrizione
Questa repository contiene progetti ed esercitazioni con Arduino IDE v2
La scheda Arduino utilizzata per questi progetti è:
- Elegoo Arduino UNO R3
- Kit: Elegoo UNO R3 Advanced Starter Kit
### Struttura della Repository
La repository è organizzata in questo modo:
- Cartelle contententi i codici sorgente dei progetti
- Cartella "libraries" per le librerie utilizzate nei progetti (consigliata installazione manuale per evitare problemi)
### Come usare i progetti
1) Scaricare la cartella del progetto
   `git clone https://github.com/TomRiddle901/Arduino/tree/master/libraries` (per la cartella delle librerie)
2) Avviare Arduino IDE v2
3) Montare il circuito elettronico (i pin dove inserire l'elettronica sono scritti nel codice)
   e collegare la scheda Arduino al computer tramite USB
4) Selezionare la scheda (e la porta) sull'IDE
5) Cliccare sul tasto `Verifica/Check` in modo da controllare se ci sono errori nel codice
6) Se non ci sono errori nel codice, cliccare `Carica/Upload`
7) Per visualizzare gli output di Arduino:
   - Strumenti > Monitor Seriale
   - Combinazioni tasti: Ctrl + Shift (Maiusc) + M
